
Закон Ома для участка цепи

    U = I * R
    I = U / R
    R = U / I

Источник постоянного тока

    E    : ЭДС
    U    : напряжение
    Rвнт : внутреннее сопротивление источника питания

Потенциал

    единица измерения: Ватт
    измеряется относительно общей точки = точки нулевого потенциала (чаще всего относительно земли)
    корректно говорить: "напряжение" в точке потенциала, "падение напряжения" на сопротивлении
    потенциалы (напряжения) всех точек, между которыми нет источника ЭДС и/или сопротивления = равны

Закон ома для полной цепи

    в полной цепи источник питания и суммарная нагрузка соединены последовательно
    под нагрузкой понимается ток цепи (чем сильнее ток, тем выше нагрузка)
    U = E - I * Rвнт
        I зависит от Rвнш  =>  U питания также зависит от Rвнш
        Rвнш [ up ]  =>  I [down] , (I * Rвнт) [down]  =>  U питания [ up ]
        Rвнш [down]  =>  I [ up ] , (I * Rвнт) [ up ]  =>  U питания [down]

1 закон Кирхгофа

    Iвх1 + ... IвхK = Iис1 + ... + IисN
    Iвх = сила тока, входящего в узел цепи
    Iис = сила тока, исходящего из узла цепи

2 закон Кирхгофа

    E1 + ... + EK = U1 + ... + UN
    E = ЭДС, создающая ток в замкнутом конутре
    U = падение напряжения на сопротивлении, входящем в замкнутый контур 

Сопротивление при ...

    последовательном соединении: Rобщ = R1 + ... + RN
    параллельном соединеним: 1/Rобщ = 1/R1 + ... + 1/RN

Мощность

    P = U * I
    P = U^2 / R
    P = I^2 * R

Проводники, которые имеют полярность:
    анод = положительный вывод (к +)
    катод = отрицательный вывод (к -)
