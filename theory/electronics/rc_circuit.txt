Простейшая схема RC-цепи на постонном токе:
последовательно:
    + источника питания         Ui = напряжение
    ключ (разомкнут)                       
    резистор                    R = сопротивление
    конденсатор                 С = емкость (изначально разряжен)
    - источника питания

Протекающие процессы при замыкании ключа:

    1. если резистор отсутствует, остается только:
        - внутреннее сопротивление источника питания
        - сопротивление соединительных элементов
    так как в общем случае эквивалентное сопротивление крайне мало, его можно считать равным нулю.
    Следовательно, при замыкании ключа:
        - произойдет короткое замыкание
        - конденсатор зарядится мгновенно
        - короткое замыкание так же мгновенно окончится
    В реальности короткое замыкание будет иметь некоторую продолжительность в виду
    упомянутого выше эквивалентного сопротивления.

    2. если резистор присутствует:
        - в момент замыкания ключа в цепи появляется ток I = Ui/R так как конденсатор еще не имеет никакого заряда
        - с течением времени конденсатор накапливает заряд, что означает рост напряжения на нем и уменьшение силы тока в цепи
        - когда конденсатор полностью заряжен, ток в цепи исчезает

Из описания процеса 2. очевидно, что:
    напряжение на конденсаторе
    ток в цепи
являются функциями времени. 
Вывод формулы напряжения при заряде конденсатора:

    I = U / R , U = Ui - Uc
    I = (Ui - Uc) / R
    
    C = Q / U , Q = I * t  >>  C = (I * t) / U  
    I = (U * C) / t   >>  дифференциал по t  >>  I = (dU * C) / dt

    I = (Ui - Uc) / R , I = (dU * C) / dt
    (Ui - Uc) / R = (dU * C) / dt  >>  (Ui - Uc) = (R * C * dU) / dt   (решение дифференциала оставляем за скобками)
    Ui - Uc = -A/e^[t/(R*C)]  >>  Uc = Ui + A/e^[t/(R*C)]

    рассмотрим момент времени t = 0: есть Ui, но Uc = 0
    0 = Ui + A/e^[0/(R*C)] = Ui + A/e^0 = Ui + A
    A = -Ui
    таким образом:

    Uc = Ui * (1 - 1/e^[t/(R*C)])       [!]

При разряде конденсатора и отсутствии внешнего источник напряжения в цепи 
(конденсатор замкнут на нагрузку) просто исчезает один элемент уравнения:

    I = -U / R   (минус, так как разряд)
    I = (dU * C) / dt
    -Uc / R = (dU * C) / dt  >>  Uc  R = - (R * C * dU) / dt

    Uc = A/e^[t/(R*C)]                  [!]
